Стек: Django/py, jQuery/js.
Необходимо доработать задачи в веб-приложении.
1. Необходимо доработать часть фильтров. Уже есть карта на API Яндекс.Карт.
Часть фильтров по карте работает, часть не работает.
По результату фильтрации должны быть выбраны все объекты на карте, соответствующие выбранному фильтру.

Нужно добавить тоггл на странице, по которому будет включаться/выключаться функционал.
При включенном функционале необходимо, чтобы при наведении на тег, озвучивалось его текстовое и цифровое содержимое.
3. Необходимо добавить на карту (API Яндекс Карт) движение в реальном времени объектов (общественный транспорт) из другого API (например, через API АСК https://ask-glonass.ru/).
У Яндекс.Карт такого функционала судя по всему нет.
Необходимо добавить список определенных фильтров для транспорта (аналогичному п.1.).
